Web Design & Development

BlannQ Website

A clean and responsive website designed to elevate brand presence through modern UI and intuitive user experience.

PROJECT TYPE

Personal Project

TIMELINE

2 Weeks

ROLE

UI/UX Design, Development

YEAR

2025

Project Overview

BLANNQ is a modern personal web design project created to highlight design versatility and frontend development skills. The goal was to build a responsive and visually appealing site that could work as a design showcase or personal brand platform.

This website focuses on layout aesthetics, accessibility, and simplicity, aimed at creating a smooth browsing experience on all devices.

Homepage showcasing the clean layout and bold design of BLANNQ

My Approach

This personal project followed a full-cycle process emphasizing user-centric design and responsive development. Here's a breakdown of the workflow:

01

Planning & Inspiration

Researched modern portfolio layouts and branding styles to shape the visual direction of BLANNQ.

02

Wireframing

Outlined basic structure and content blocks to ensure clarity in layout and user flow.

03

UI Design

Developed a clean, bold UI using a neutral palette and modern fonts to reflect professionalism and creativity.

04

Development

Used HTML, CSS, and basic JavaScript to bring the design to life with responsive layouts and smooth transitions.

05

Testing & Launch

Tested across browsers and devices to ensure consistent rendering before launching via GitHub Pages.

Design System

BLANNQ uses a minimal yet bold design system with emphasis on readability, hierarchy, and color balance.

Technology Stack

HTML CSS JavaScript

Key Features

This project features elements that demonstrate effective frontend design principles:

  • Responsive Design – Mobile-friendly and adaptable layouts for different screen sizes
  • Bold Typography – Visually striking fonts to communicate clarity and intent
  • Clean Navigation – Easy-to-use menu structure for intuitive user interaction
